# HTTP Methods

| Method | Meaning                                           |
| ------ | ------------------------------------------------- |
| GET    | get data from server without changine the origins |
| POST   | create a new record                               |
| PUT    | override / Update a complete record               |
| PATCH  | update one part of record                         |
| DELETE | Delete resources                                  |
