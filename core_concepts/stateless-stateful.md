# Stateless and Stateful

| Feature      | Stateless                    | Stateful                          |
| ------------ | ---------------------------- | --------------------------------- |
| Data         | No data stored on the server | Session data stored on the server |
| Independence | Requests are independent     | Requests depend on each other     |
| Scalability  | Very easy (scale-out)        | Difficult (tied to a server)      |
| Examples     | HTTP, REST, DNS              | FTP, TCP, Database                |
